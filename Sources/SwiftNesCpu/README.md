# nes-cpu

NES CPU

# Install

```
$ brew tap takuyaohashi/SwiftNes https://github.com/takuyaohashi/SwiftNes
$ brew install SwiftNes
```

# Usage

```
$ nes-cpu nes_file 
```

## Commands

|commands    |mean                     |
|------------|-------------------------|
| s or step  |exec current instruction |
| r or reg   |show current register    |
| q or quit  |quit this app            |
