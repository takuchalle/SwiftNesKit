# nes-disasm

NES Disassember

# Install

```
$ brew tap takuyaohashi/SwiftNesKit https://github.com/takuyaohashi/SwiftNesKit
$ brew install SwiftNesKit
```

# Usage

```
$ nes-disasm nes_file 
```
